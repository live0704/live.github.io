🎮 내 그래픽카드 확인하기

아래 버튼을 눌러 현재 사용 중인 그래픽카드 정보를 확인하세요.

💡 정확한 그래픽카드 확인 방법 (초보자용)

이 방법은 누구나 쉽게 따라 할 수 있어요!

  1. 키보드에서 Ctrl + Shift + Esc 키를 동시에 눌러 **작업 관리자**를 엽니다.
  2. 상단 탭에서 **성능**을 클릭합니다.
  3. 왼쪽 목록에서 **GPU 0** 또는 **GPU 1**을 선택합니다.
  4. 오른쪽 상단에 **그래픽카드 모델명**이 표시됩니다!

💡 노트북이라면 **GPU 0 = 내장 그래픽**, **GPU 1 = 외장 그래픽** 입니다.

⚡ 고급 확인 방법 (dxdiag 사용)

아래 명령어를 복사하여 Windows 실행창에서 입력하면 상세 정보를 볼 수 있습니다.

  1. 키보드에서 Windows + R 키를 동시에 눌러 실행창을 엽니다.
  2. 복사한 명령어를 실행창에 붙여넣기 (Ctrl + V)
  3. 확인을 누르면 "DirectX 진단 도구"가 실행됩니다.
  4. 상단 탭에서 **디스플레이(Display)**를 클릭하면 그래픽카드 정보를 확인할 수 있습니다.
